## Getting Started

### Account Registration

1. **Visit the Registration Page**
   - Navigate to the registration page from the homepage
   - Choose your role: Buyer, Owner, or Agent

2. **Fill in Your Details**
   - Enter your name, email, mobile number, and password
   - Select your appropriate role
   - Agree to the terms and conditions

3. **Verify Your Account**
   - After registration, you'll receive an OTP on your registered mobile number
   - Enter the OTP to verify your account
   - You'll be automatically logged in after verification

### Login Process

1. **Enter Credentials**
   - Provide your email and password
   - Click "Login"

2. **OTP Verification**
   - An OTP will be sent to your registered mobile number
   - Enter the OTP to complete login

3. **Dashboard Access**
   - After successful verification, you'll be redirected to your role-specific dashboard

## Admin Guide

### Overview
As an admin, you have full control over the platform, including user management, property approvals, and system settings.

### Dashboard
- **Overview Statistics**: View total users, properties, and system metrics
- **Recent Activity**: Monitor recent property submissions and user registrations
- **Pending Approvals**: Quick access to properties awaiting approval

### User Management

#### Viewing Users
1. Navigate to "Users" in the sidebar
2. View paginated list of all users
3. Filter users by role (Admin, Owner, Agent, Buyer)
4. Search users by name or email

#### Managing Users
- **Edit User**: Click the edit button to modify user details
- **Change Role**: Update user roles as needed
- **Assign Plans**: Assign or change subscription plans
- **Delete User**: Remove users (use with caution)

### Property Management

#### Property Overview
1. Go to "Properties" section
2. View all properties with their current status
3. See property versions and approval status

#### Version Control
- **View Versions**: See all versions of a property
- **Compare Changes**: Use the diff tool to see what changed between versions
- **Approve/Reject**: Approve valid changes or reject inappropriate modifications

#### Bulk Operations
- **Bulk Approve**: Select multiple versions for approval
- **Bulk Reject**: Reject multiple versions at once

### Settings Management

#### System Settings
1. Access "Settings" from the sidebar
2. Configure:
   - Site name and contact information
   - Default plan limits
   - System-wide configurations

#### Plan Management
- View existing plans and their capabilities
- Modify plan features and pricing
- Add new plans or deactivate existing ones

### Todo Management
- Create system-wide tasks
- Assign tasks to different user roles
- Track completion status
- Set due dates for important tasks

## Troubleshooting

### Login Issues

#### OTP Not Received
- Check if mobile number is correct
- Ensure good network connectivity
- Try requesting OTP again after 5 minutes
- Contact support if issue persists

#### Account Locked
- Contact admin for account unlock
- Verify identity information

### Payment Issues

#### Payment Failed
- Check payment method details
- Ensure sufficient balance
- Try different payment method
- Contact support with transaction ID

#### Payment Not Reflected
- Wait 5-10 minutes for processing
- Check payment confirmation email
- Contact support with payment details

### Property Issues

#### Property Not Approved
- Check admin feedback on rejection
- Make necessary corrections
- Resubmit for approval

#### Contact Access Denied
- Check your plan limits
- Upgrade plan or make payment for additional access
- Contact support if limits seem incorrect

### Technical Issues

#### Page Not Loading
- Clear browser cache
- Try different browser
- Check internet connection
- Contact support with error details

#### Form Submission Errors
- Check all required fields are filled
- Verify data format (email, phone, etc.)
- Ensure file uploads meet size limits
